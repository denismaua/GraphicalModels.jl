GraphicalModels.FactorGraph — Type

A Factor Graph is a bipartite graph where nodes are either variables or factors.

Arguments

  • variables: vector of variable nodes.
  • factors: vector of factor nodes.
source
GraphicalModels.FactorGraph — Method

Reads a model from file in UAI Competition Format and returns the correspoding factor graph. See https://www.cs.huji.ac.il/project/UAI10/fileFormat.php for details about the file format.
